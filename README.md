**Introduction**
Dart is a modern programming language developed by Google. It is mainly used for building applications with Flutter. Dart is designed to be easy to learn and efficient for developing fast and responsive applications.
Topics
**1. What is Dart, and why is it used in Flutter?**
Dart is a programming language created by Google. It is widely used with Flutter for developing mobile, web, and desktop applications.
Reasons why Dart is used in Flutter:
•	It provides high performance.
•	It supports Hot Reload, which allows developers to see changes instantly.
•	It works efficiently for building user interfaces.
•	It is easy to learn and similar to Java and other C-style languages.
**2. Main Data Types in Dart**
Data types are used to define the type of data that a variable can store.
Data Type	Description	Example
int	Stores whole numbers	10, 25
double	Stores decimal numbers	10.5
String	Stores text	"Hello"
bool	Stores true or false	true
List	Stores multiple values like an array	[1,2,3]
Map	Stores data in key–value pairs	{"name":"Ali"}
var	Automatically determines data type	var age = 20
dynamic	Data type can change during runtime	dynamic x = 10

**3. Types of Operators in Dart**
Operators are symbols used to perform operations on variables and values.
Operator Type	Operators	Purpose
Arithmetic	+ - * / % 	Used for mathematical calculations
Relational	== != > < >= <=	Used to compare values
Logical	&& || !	Used for logical conditions
Assignment	= += -= *= /=	Used to assign values to variables
Unary	++ -- 	Increment, decrement, negative, logical NOT
Ternary	condition? a: b	Short form of if–else
Type Test	is is!	Used to check the data type
